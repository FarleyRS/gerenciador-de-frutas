Projeto de gerenciamento de frutas sem integração com banco de dados
################
Desenvolvido em Java 
by Farley Ramos de Sena
ano 2020 2° perido

O projeto de gerenciamento tem as seguintes funcionalidades:
    #Inserir fruta
    #Excluir fruta 
    #Listar frutas 
    #Calcular pesos das frutas
    #Calcular média de calorias das frutas
    #Mostrar os dados de uma fruta
    #Mostrar a fruta com menor calorias
    #Mostrar a fruta com maior peso médio
